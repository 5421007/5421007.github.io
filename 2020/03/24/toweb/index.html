Github+Hexo搭建网站

发表于 2020-03-24

Hexo简介

Hexo 是一个基于 Nodejs 的静态博客网站生成器，作者是来自台湾的 Tommy Chen(https://zespia.tw/)

特点

不可思议的快速 ─ 只要一眨眼静态文件即生成完成
支持 Markdown
仅需一道指令即可部署到 GitHub Pages 和 Heroku
已移植 Octopress 插件
高扩展性、自订性
兼容于 Windows, Mac & Linux

Next 是 iissnan 在 GitHub 上开源的一个 Hexo 主题，主打简洁。
注意之后我们要用到两个名字相同位置不同的.yml文件。在 Hexo 中有两份主要的配置文件，其名称都是 _config.yml
。 其中，一份位于站点根目录下（即 xxx.github.io 目录下），主要包含 Hexo 本身的配置；另一份位于主题目录下(即 xxx.github.io 目录下生成的theme文件夹中)。这份配置有主题作者提供，主要用于配置主题相关的选项。
